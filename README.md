# Lookg

A lightweight, single-page psychoeducational cards web app focused on confidence, communication, productivity, and personal growth.

## Features
- Search with autocomplete suggestions
- Theme filters, quick filters, and sorting
- Accessible modal with keyboard navigation
- Reading mode + text size controls
- Saved cards and “continue last card”
- GitHub Pages ready

## Run locally
Open `index.html` in your browser.
